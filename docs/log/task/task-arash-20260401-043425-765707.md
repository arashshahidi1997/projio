# Flatten pipe/flow: update projio wrappers and MCP server registration

## Goal

Update the projio-side MCP wrappers (`src/projio/mcp/pipeio.py`) and server tool registrations (`src/projio/mcp/server.py`) to match the flow-only API from the prior tasks.

## Context

Prior tasks changed:
- Registry: flow-only keys, `derivative` instead of `pipe`
- pipeio MCP functions: all dropped `pipe` param
- pipeio CLI: all dropped `pipe` positional arg

This task updates the projio layer that wraps pipeio's MCP functions and exposes them to AI agents.

## Prompt

> Update the projio MCP wrappers and server registration for flow-only addressing.
>
> ### Files to modify
>
> **`src/projio/mcp/pipeio.py`:**

> return json_dict(mcp_nb_sync(root, flow=flow, name=name, ...))
