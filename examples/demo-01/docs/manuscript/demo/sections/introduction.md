# Introduction

The observation that word frequencies in natural language follow a power-law
distribution is among the most robust empirical findings in linguistics
[@zipf1949]. Despite decades of research, the precise mechanisms underlying
this regularity remain debated.

In this work, we revisit Zipf's law using a modern text analysis pipeline.
